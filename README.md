# Vents-OS

Vents OS is a web-based operating system (currently v0.9 alpha) that I’m working on.

Here are some of the features:
- A web browser
- File saving
- A straightforward installation process
- 3 small built-in games
- Built-in chatbot
- Calendar & calculator
- A settings app
- A task manager
- A terminal

## Requirements
- 180 MB storage
- 64 MB RAM
- Modern browser

## Alpha version
This is an early alpha build, so expect bugs, glitches, and unfinished parts. I’m still fixing and improving things as I go.
